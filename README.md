# YT Strategy Dashboard

Auto-generated daily bias dashboards from YouTube trader videos.

- Live dashboard: https://maddhatter24d-sketch.github.io/yt-strategy-dashboard/
- Code that generates these: not in this repo (the generator runs locally on the author's machine; only the rendered output ships here).

These dashboards summarise **what YouTube channels SAY** about market direction.
They are a confluence input, **not** trade signals.
